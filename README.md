# EvilESP32
EvilESP32 是一个基于 ESP32 的 BLE 广播模拟器，能够生成 Apple Continuity、Google Fast Pair、Samsung Easy Setup、Microsoft Swift Pair 以及 Lovespouse 等主流厂商的专属广播数据包。当手机/电脑等终端设备扫描到这些广播时，会触发系统级弹窗（如 AirPods 连接提示、Fast Pair 配对通知、Windows 快速配对等）。  本项目仅供安全研究、蓝牙协议教学和渗透测试授权场景使用。通过剖析商业 BLE 广播协议，开发者可以深入了解各厂商的弹窗触发机制，并为设备兼容性测试、企业安全演练提供工具支持。
